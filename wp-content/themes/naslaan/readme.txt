== Installation ==

1. Navigate to "Appearance" => "Themes" => "Add New", and click the "Upload Theme" button.
2. Navigate to find the zipped file on your computer and hit "Install Now".
3. Activate the installed theme by going to "Appearance" => "Themes" and clicking the "Activate" button.
4. Once you activate the theme, you will see the notification message, click the "Begin installing plugins" button to go to "Install Required Plugins" where you need to install and activate the required plugins.
5. Click the "Install" button on each plugin to start installing them and activate them after installation.
6. After Unyson WordPress Framework plugin activation, you will have notification to install Unyson plugin extensions, click the "Install theme compatible extensions" and install all these extensions. The theme will not be fully functional without these Unyson extensions.
